Assessment Builder Options

Overview
- The builder has five steps: Start, General Info, Visual Assets, Questions, Rubrics.
- Drafts autosave on blur and periodically while typing.
- Builder is read-only once an assessment is no longer in draft status.

Start (Step 1)
- Authoring mode:
  - Start from scratch (manual).
  - Upload a PDF to generate a draft.
  - Generate with AI from a prompt.
- Class selection (required).
- Assessment title (required).
- PDF upload flow:
  - Upload a PDF (not stored).
  - Choose number of questions (1-5).
  - Generates title, questions, and both rubrics.
- AI generation flow:
  - Provide a prompt (min 20 characters).
  - Choose number of questions (1-5).
  - Generates title, questions, rubrics, and an image prompt.

General Info (Step 2)
- Subject selection (History, Science, Literature, Math, Language Arts, Other).
- Target language selection (English US, Spanish, French, Mandarin, Arabic, Other).
- Student instructions (up to 500 characters).

Visual Assets (Step 3)
- Optional anchor image for the assessment.
- Generate images from a prompt (creates 3 options).
- Paste a public image URL instead of generating.
- Preview selected image.
- Clear image selection.

Questions (Step 4)
- Add, edit, and delete questions.
- Question types (friendly labels):
  - Open response.
  - Evidence follow-up (photo).
  - Evidence follow-up (artifact alias).
  - Audio follow-up (AI question).
  - Custom type (free text).
- Evidence upload setting per question:
  - Disabled, Optional, or Required.
  - Evidence follow-up types force evidence to Required.
- Evidence upload supports photo-based evidence (student uploads before recording).

Rubrics (Step 5)
- Dual scorer setup with separate rubrics:
  - Scorer 1: Reasoning and synthesis.
  - Scorer 2: Evidence and factual accuracy.
- Instructions text per rubric (required).
- Scale min and max (1-5).
- Publish readiness requires both rubrics to be complete.

Integrity Shields (Global)
- Pausing guardrail (flags silence over 2.5s).
- Focus monitor (tracks tab switching).
- Dynamic shuffle (randomizes question order per student).
- Academic integrity pledge:
  - Toggle on/off.
  - Optional custom pledge text (defaults if blank).

Recording Limits (Global)
- Recording limit per response: 30s, 60s, 90s, 120s, 180s.
- Viewing timer for retell: 10s, 15s, 20s, 30s.
